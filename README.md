## What Is AAC?

**AAC** stands for **Augmentative and Alternative Communication**.

AAC includes methods and tools that help a person communicate when speaking is difficult or when their speech is not easily understood.

* **Augmentative** communication adds support to a person’s existing speech.
* **Alternative** communication provides another method when speech cannot be used.

AAC can include:

* Gestures
* Facial expressions
* Written words
* Picture cards
* Communication boards
* Symbols
* Mobile applications
* Speech-generating devices
* Text-to-Speech technology

AAC can be used temporarily or permanently, depending on the individual’s communication needs.

---

## How AAC Works

A typical AAC application allows a user to select pictures, symbols, or words that represent what they want to communicate.

For example, the user may select:

```text
I → want → water
```

The application creates the sentence:

```text
I want water.
```

The user can then press the speak button, and the application reads the sentence aloud using Text-to-Speech technology.

AAC helps the user communicate with:

* Parents
* Family members
* Teachers
* Caregivers
* Therapists
* Doctors
* Friends
* Members of the community

## About Dhvani

**Dhvani** is an Augmentative and Alternative Communication application designed for people who have difficulty communicating through speech.

The word **Dhvani** means **sound** or **voice**. The purpose of the application is to give users a simple and supportive way to express themselves using words, symbols, categories, emotions, routines, and spoken messages.

A user can select one or more communication cards, create a message, and use the application to speak that message aloud.

Dhvani is designed for children and adults who may benefit from additional communication support, including people with autism, cerebral palsy, apraxia, developmental disabilities, acquired communication difficulties, or temporary speech limitations.

---


## How Dhvani Works

Dhvani provides a clear communication process:

```text
Open Dhvani
      ↓
Select a word or category
      ↓
Add words to the sentence area
      ↓
Review the complete message
      ↓
Press the Speak button
      ↓
Dhvani speaks the message aloud
```

### Example

The user selects:

```text
I need → help
```

Dhvani builds and speaks:

```text
I need help.
```

The user can add words, remove words, clear the message, or speak the completed sentence.

Users can express feelings such as:

* Tired

* Worried

Emotional communication is important because communication should include more than basic requests such as food, water, or help.

### 6. Calm Corner

The **Calm Corner** provides a dedicated area that can support users during moments of stress, sensory overload, anxiety, or emotional discomfort.

This section can help users identify calming actions and communicate what support they need.

The Calm Corner makes Dhvani more than a basic word-selection tool by connecting communication with emotional regulation.

* Lunch

### 8. Visual Communication Cards

Each communication card combines:

* A recognizable symbol

---

### 9. Text-to-Speech

After building a message, users can make Dhvani speak it aloud.

Dhvani uses bottom navigation to provide access to its main sections:

* Home

## What Makes Dhvani Different?

Several established AAC applications provide large vocabularies, customization, language-learning systems, and advanced communication tools. Dhvani is not intended to copy or replace those applications.

Dhvani is being developed with its own focused experience.

### Simple Starting Experience

Some AAC systems can feel difficult for a new user because they display many words, folders, controls, and configuration options at the same time.

Dhvani begins with:

* Common quick-access phrases
* Clearly separated categories
* A visible sentence area
* Simple bottom navigation
* Large communication cards

This can help new users and caregivers understand the basic interaction more quickly.

## Accessibility Principles

Dhvani should be developed according to the following principles:

* Communication buttons must be easy to identify.
* Touch targets must be large enough for mobile use.
* Important functions should not depend only on color.
* Text should remain readable at larger display sizes.
* Screen-reader users should receive meaningful labels.
* Keyboard users should be able to navigate the application.
* Controls should not overlap on smaller screens.
* Accidental selections should be minimized.
* Users should be able to undo or clear selections.
* Communication should remain available with as few steps as possible.

## Upcoming Features

**1. Tabula Rasa Mode**

**"Every communication journey starts with a blank slate."**

The term Tabula Rasa means "blank slate." In Dhvani, this mode allows users, caregivers, or therapists to build a completely personalized communication system from scratch instead of starting with prebuilt boards.

**Purpose**

Every AAC user communicates differently. Rather than forcing everyone to use the same vocabulary, Tabula Rasa Mode lets communication grow with the individual.
As the child develops, additional vocabulary can gradually be added without overwhelming them.

**2. Sherlock Mode**

**"Helping caregivers understand communication patterns."**

Sherlock Mode is an intelligent observation and pattern analysis feature.

Instead of only recording what users say, Sherlock Mode analyzes communication behavior to help caregivers and therapists discover useful insights.

**Purpose**

Identify communication trends that may not be obvious during daily use.
This information can help caregivers understand needs and adapt communication boards.

**3. Experiment Journal**

**"Track communication growth, one step at a time."**

Experiment Journal is a structured log where therapists, teachers, caregivers, and researchers can document observations, communication sessions, and progress over time.

It is designed to support evidence-based improvements to a user's AAC experience.

**Purpose**

Keep a history of communication experiments, observations, and successful strategies.
